漫トロWeb
====

## Description
http://mantropy.net/ のソースコードです。

楽天書籍APIから書籍情報を取得し、漫画の一覧を表示します。漫画ランキングを作ることもできます。

バグ、機能改善案等があればIssueを立ててください

## Install
`git clone git@github.com:hidesys/mantropy.git`してソースコードをcloneしてください。

その後、
* `.env.sample`
をコピーして
* `.env.development`
を作成し、必要となる環境変数をセットしてください。
